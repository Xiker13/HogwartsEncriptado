Hogwarts Encrypted Manual

Top Menu

The top menu contains the following options:

  • File → Close: Closes the application.
  • Help → Manual: Opens this manual.
  • Help → Help: Additional information about the application.
  • About → About... Information about the development team and app version.
  • Language: Allows changing the application language.
  • Theme: Changes the interface theme (Classic, Hogwarts Library, Slytherin).

Algorithm Selection

Choose the type of encryption you want to use:

  • AES: Advanced symmetric encryption algorithm.
  • Vigenère: Text substitution encryption algorithm.

Key

Enter the key to be used for encryption or decryption:

  • Type the key directly into the text field.
  • Or click "Load key" to select a file containing the key.

Main Area

It is divided into two sections: input and result.

  • Input: Type the text to encrypt/decrypt or load a file with the content.
  • Result: The encrypted or decrypted text will appear here.
  • To save the result, enter the path and click "Save".

Encrypt / Decrypt Buttons

  • Encrypt: Applies the selected algorithm to the input and shows the result.
  • Decrypt: Reverts the encryption and shows the original text.

Image Mode

To encrypt or decrypt images (only available using AES):

  • Enable the Image option in the "Save as" section.
  • Enter or load the key as usual.
  • Click Encrypt or Decrypt.
  • You will first be prompted to select the image file to process.
  • Then you will be asked where to save the resulting image.
  • After completion, a success message will appear: "Image encrypted successfully" or "Image decrypted successfully".

Common Errors

  • Key required: No key was entered.
  • Empty text: No text was entered.
  • Encryption/Decryption error: There was a problem processing the file or text.
  • Nothing to save: Cannot save if there is no result.
  • Manual not found: Could not locate the manual file.
