Top Menu

The top menu contains the following options:

  • File → Close: Closes the application.
  • Help → Manual: Opens this manual.
  • Help → Help: Additional information about the application.
  • About → About Us: Information about the development team.

Algorithm Selection

Choose the type of encryption you want to use:

  • AES: Advanced symmetric encryption algorithm.
  • Vigenère: Text substitution encryption algorithm.

Key

Enter the key to use for encryption or decryption:

  • Type the key directly into the text field.
  • Or click "Load File" to select a file containing the key.

Main Area

It is divided into two sections: input and result.

  • Input: Enter the text to encrypt/decrypt or load a file with the content.
  • Result: The encrypted or decrypted text will appear here.
  • To save the result, enter the filename and click "Save File".

Encrypt / Decrypt Buttons

  • Encrypt: Applies the selected algorithm to the input and displays the result.
  • Decrypt: Reverses the encryption using the selected algorithm and shows the original text.
