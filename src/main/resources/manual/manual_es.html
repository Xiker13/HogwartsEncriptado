Menú superior

El menú superior contiene las siguientes opciones:

  • Archivo → Cerrar: Cierra la aplicación.
  • Ayuda → Manual: Abre este manual.
  • Ayuda → Ayuda: Información adicional sobre la aplicación.
  • Sobre Nosotros → Acerca de... Información sobre el equipo desarrollador.

Selección de algoritmo

Elige el tipo de cifrado que deseas usar:

  • AES: Algoritmo de cifrado avanzado simétrico.
  • Vigenère: Algoritmo de cifrado por sustitución de texto.

Clave

Introduce la clave que se usará para cifrar o descifrar:

  • Escribe la clave directamente en el campo de texto.
  • O haz clic en "Cargar archivo" para seleccionar un archivo que contenga la clave.

Área principal

Se divide en dos secciones: entrada y resultado.

  • Entrada: Escribe el texto a cifrar/descifrar o carga un archivo con el contenido.
  • Resultado: Se mostrará aquí el texto cifrado o descifrado.
  • Para guardar el resultado, introduce la ruta y haz clic en "Guardar archivo".

Botones Cifrar / Descifrar

  • Cifrar: Aplica el algoritmo seleccionado a la entrada y muestra el resultado.
  • Descifrar: Aplica el algoritmo seleccionado para revertir el cifrado y muestra el texto original.
