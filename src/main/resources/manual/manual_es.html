Menú superior

El menú superior contiene las siguientes opciones:

  • Archivo → Cerrar: Cierra la aplicación.
  • Ayuda → Manual: Abre este manual.
  • Ayuda → Ayuda: Información adicional sobre la aplicación.
  • Sobre → Acerca de... Información sobre el equipo desarrollador y versión de la app.
  • Idioma: Permite cambiar el idioma de la aplicación.
  • Tema: Cambia el tema de la interfaz (Básico, Biblioteca Hogwarts, Slytherin).

Selección de algoritmo

Elige el tipo de cifrado que deseas usar:

  • AES: Algoritmo de cifrado simétrico avanzado.
  • Vigenère: Algoritmo de cifrado por sustitución de texto.

Clave

Introduce la clave que se usará para cifrar o descifrar:

  • Escribe la clave directamente en el campo de texto.
  • O haz clic en "Cargar clave" para seleccionar un archivo que contenga la clave.

Área principal

Se divide en dos secciones: entrada y resultado.

  • Entrada: Escribe el texto a cifrar/descifrar o carga un archivo con el contenido.
  • Resultado: Se mostrará aquí el texto cifrado o descifrado.
  • Para guardar el resultado, introduce la ruta y haz clic en "Guardar".

Botones Cifrar / Descifrar

  • Cifrar: Aplica el algoritmo seleccionado a la entrada y muestra el resultado.
  • Descifrar: Revierten el cifrado aplicado y muestran el texto original.

Modo Imagen

Para cifrar o descifrar imágenes (solo disponible usando AES):

  • Activa la opción Imagen en la sección "Guardar como".
  • Ingresa o carga la clave como de costumbre.
  • Haz clic en Cifrar o Descifrar.
  • Primero se te pedirá seleccionar el archivo de imagen a procesar.
  • Después se te pedirá indicar dónde guardar la imagen resultante.
  • Una vez terminado, se mostrará un mensaje de éxito: "Imagen cifrada correctamente" o "Imagen descifrada correctamente".

Errores comunes

  • Clave requerida: No se ha ingresado clave.
  • Texto vacío: No se ha ingresado texto.
  • Error al cifrar/descifrar: Ocurrió un problema al procesar el archivo o texto.
  • No hay nada que guardar: No se puede guardar si no hay resultado.
  • Manual no encontrado: No se pudo localizar el archivo del manual.
